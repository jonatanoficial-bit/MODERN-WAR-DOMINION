# Bandeiras do Modern War Dominion

Coloque aqui as bandeiras do seu pacote local.

Padrão que o jogo reconhece automaticamente:
- `br.png` para Brasil
- `us.png` para Estados Unidos
- `cn.png` para China
- `ru.png` para Rússia
- `gb.png` ou `uk.png` para Reino Unido
- `ua.png` para Ucrânia

O jogo tenta nesta ordem: `.png`, `.webp`, `.jpg`, `.jpeg`, `.svg`, `.gif`.
Os nomes devem ficar em minúsculo e com código ISO de 2 letras, como aparecem nos seus prints: `br`, `ca`, `cn`, `il`, `jp`, `kr`, etc.

A pasta correta no projeto é:
`assets/flags/`
