# Fase 32 — Estado-Maior, Comandantes e Doutrina

A Fase 32 adiciona uma camada de comando profissional.

## Sistemas
- Estado-Maior
- Comandantes por função
- Doutrina militar
- Moral do Estado-Maior
- Fadiga de comando
- Ordens estratégicas
- Bônus por especialidade

## Gameplay
O jogador passa a tomar decisões de alto comando: escolhe doutrina, atribui oficiais, reorganiza Estado-Maior e gerencia fadiga/moral. Isso melhora ataques, logística, inteligência e defesa de forma sistêmica.
