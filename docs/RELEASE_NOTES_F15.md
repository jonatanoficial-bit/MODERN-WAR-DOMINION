# Fase 15 — Sistema trilíngue base PT/EN/ES

Esta fase inicia a internacionalização real do Modern War Dominion.

## Idiomas
- Português do Brasil (`pt-BR`)
- Inglês (`en-US`)
- Espanhol (`es-ES`)

## Arquivos
- `data/lang/pt-BR.json`
- `data/lang/en-US.json`
- `data/lang/es-ES.json`

## Implementado
- Seletor de idioma no topo.
- Persistência do idioma no navegador.
- Tradução da interface principal.
- Tradução do Quartel-General Mobile.
- Tradução de abas, botões, textos da home, mapa e operações.
