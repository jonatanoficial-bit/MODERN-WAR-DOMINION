# Fase 16 — Tutorial guiado e missões iniciais

A Fase 16 melhora a jogabilidade mobile com um tutorial de campanha dentro do Quartel-General.

## Missões iniciais
1. Construir primeira base
2. Concluir construção
3. Produzir primeira unidade
4. Receber tropa operacional
5. Fazer reconhecimento
6. Sobreviver aos 3 primeiros meses

Cada missão dá recompensa e reduz a sensação de jogo confuso.
