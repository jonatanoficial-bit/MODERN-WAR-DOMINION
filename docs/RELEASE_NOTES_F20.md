# Fase 20 — Guerra Terrestre e Ocupação

A Fase 20 transforma a guerra em conquista territorial.

## Sistemas
- Frente terrestre
- Avanço territorial
- Suprimento
- Resistência local
- Baixas
- Ocupação
- Pacificação
- Retirada de frente
- Marcadores de frente no mapa real

## Gameplay
O jogador não apenas ataca: agora ele abre uma frente, reforça o avanço, sofre desgaste, ocupa território e precisa reduzir resistência para manter a conquista.
