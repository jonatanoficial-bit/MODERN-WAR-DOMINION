# Fase 17.1 — Hotfix seleção de país

Correção focada na tela de seleção de nação.

## Corrigido
- Botão de confirmar país não aparecia no PC/mobile por cards encavalados.
- Cards de países com altura insuficiente.
- Grid de países sem espaço inferior suficiente.

## Novo comportamento
- Toque/clique no card para selecionar o país.
- Um painel fixo inferior mostra a bandeira e o país selecionado.
- O botão `Confirmar país` fica sempre visível.
