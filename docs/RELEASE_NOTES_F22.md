# Fase 22 — Guerra Naval

A Fase 22 adiciona a camada naval estratégica ao jogo.

## Sistemas
- Controle marítimo
- Pressão de bloqueio
- Ameaça submarina
- Alcance de porta-aviões
- Segurança de comboios
- Patrulha naval
- Bloqueio naval
- Ataque submarino
- Ataque de porta-aviões
- Escolta de comboios
- Marcadores de operações navais no mapa real

## Gameplay
O jogador agora pode proteger rotas comerciais, bloquear rivais, usar submarinos, projetar poder com porta-aviões e lidar com ameaça marítima inimiga.
