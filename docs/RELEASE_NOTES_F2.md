# Release Notes — Fase 2

A Fase 2 transforma Modern War Dominion em um simulador global inicial. O jogador escolhe uma nação com bandeira, analisa dados de jogo, observa o mapa mundial e toma decisões mensais.

## Destaques

1. Mapa mundial gratuito estilo Leaflet/OpenStreetMap.
2. Regiões globais clicáveis.
3. Países com bandeiras e estatísticas ampliadas.
4. Ações mensais e eventos globais.
5. Save local com chave nova.

## Limite consciente

Os dados são índices de simulação/game design, não base oficial fechada. Servem para equilíbrio inicial do jogo.
